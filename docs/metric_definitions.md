# Metric Definitions

## Bed Occupancy Rate

Percentage of available bed-days occupied by admitted patients in the selected period.

## Average Length Of Stay

Average number of inpatient days between admission and discharge.

## 30-Day Readmission Rate

Percentage of discharged encounters flagged as an unplanned readmission within 30 days.

## Hospital-Acquired Infection Rate

Number of hospital-acquired infection events per 100 inpatient encounters.

## In-Hospital Mortality Rate

Percentage of inpatient encounters ending in death.

## Emergency Waiting Time

Median number of minutes between emergency arrival and first clinician assessment.

## Target Status

Each KPI is compared against a configurable target. The status is:

- `on_target` when performance meets the quality target
- `watch` when performance is close to the threshold
- `off_target` when performance needs escalation
